# Project Ideas

A list of project ideas for developers at all levels.

## Levels
- [Beginner](#beginner)
- [Intermediate](#intermediate)
- [Advanced](#advanced)

## Beginner
- To-do list app (classic, but great for learning CRUD)
- Weather app using a free API
- Random quote generator
- Tip calculator
- Markdown previewer

## Intermediate
- URL shortener with a backend
- Budget tracker with charts
- Pomodoro timer with sound
- GitHub profile viewer using the GitHub API
- Flashcard app with spaced repetition

## Advanced
- Real-time chat app with WebSockets
- Self-hosted bookmark manager
- CLI tool that does something you do manually every day
- Browser extension for tab management
- REST API with auth, rate limiting, and docs

Feel free to use any of these — no credit needed.
